*MyBakeryApp
React Application for A bakery App.

1. How to use?
    1. run:  git clone https://github.com/RahulGahalaut/MyBakeryApp.git
    2. run:  npm install
    3. run:  npm start

2. go to homepage, add item into the carts
3. click on "checkout" Button
4. click on "Pay Now" button 
5. see the logs
